Number Guessing Game - Python Project

- **Game Overview**:
  - This game challenges you to guess a randomly chosen number between 1 and 100.
  - You have 10 attempts to guess the correct number.
  - After each guess, the game gives hints: whether the guess is too high or too low, with the range of possible numbers.
  - The game ends when you guess the correct number or run out of attempts.

- **Error Handling**:
  - This script handles errors by ensuring only numeric input is accepted.

- **How to Play**:
  - Run the script, follow the prompts, and guess the number within 10 attempts.
